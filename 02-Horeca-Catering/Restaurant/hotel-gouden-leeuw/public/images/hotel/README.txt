Hotel De Gouden Leeuw - Image Assets
====================================

This folder contains the hotel's image assets organized by category.

FOLDER STRUCTURE:
- exterior/     - Hotel facade, entrance, terrace
- lobby/        - Reception, seating, staircase
- rooms/        - All room types and bathrooms
- restaurant/   - Dining room, dishes, breakfast
- spa/          - Treatment rooms, sauna, relaxation
- amenities/    - Concierge, key cards, toiletries

DOWNLOAD IMAGES:
Run the following command to download sample images from Unsplash:

    npm run download-images

Or manually:

    node scripts/download-images.js

The script will download 25 sample images suitable for the demo.

NOTE: These are placeholder images from Unsplash for demonstration purposes only.
In a production environment, replace these with actual hotel photography.

DEMO WEBSITE NOTICE:
All images are used for demonstration purposes only.
