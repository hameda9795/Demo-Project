# Van Dijk Loodgieters Website

Complete Next.js 14 website for a Dutch plumbing company in Rotterdam.

## Structure
- Homepage with 8 main sections
- Admin panel with dashboard
- Customer portal with service tracking
- Blog/knowledge base
- Privacy policy page

## Stack
Next.js 14, TypeScript, Tailwind CSS, Framer Motion

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
# Static export to /dist
```
